Semi-Auto Fitting with IRAF

Apr 13, 2016

The Method

First, we normalize with the Continuum task from the stsdas package.

Then we select the points where to fit the gaussian.

continuum ../*_* @norm.txt

Then I call a Python script and do the fit using pyraf. The script can be found in the GitHub Repo MUSE Data Reduction 2016. In the folder misc/Spectra/fitting.

The Spectra and H\(\alpha\) line

All of them:

specplot U*.fits xmin=6000 xmax=7000  label=imname graphics='eps'

This is converted to svg with:

epstopdf <file.eps>
pdf2svg <file.pdf> <out.svg>

From (http://superuser.com/questions/198460/converting-from-eps-to-svg-format)
